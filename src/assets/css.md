# css

## o que é

CSS é a camada que decide como as coisas aparecem &mdash; tipografia, cor, ritmo, repouso. Não é decoração; é hierarquia visual que guia o olhar.

## o cascateamento

O nome carrega o algoritmo. Especificidade, ordem, herança. Quando você entende a cascata, deixa de lutar contra ela.
