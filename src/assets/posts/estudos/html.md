# html

## o que é

HTML é a estrutura. É o esqueleto da página antes de qualquer estilo, antes de qualquer comportamento. Cada elemento é uma escolha semântica que carrega significado para navegadores, leitores de tela e mecanismos de busca.

## por que importa

A semântica que você escolhe agora se torna a interface de acessibilidade, o mapa do leitor de tela, e a base sobre a qual o CSS vai cair. Um botão que é uma div é uma promessa quebrada.
